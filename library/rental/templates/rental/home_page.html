Welcome to my simple library project !


This project is made fully by me in order to practice my django skills. I tried to test features that I learned from different courses on Youtube and a book that I read.
This library has:

  • Administrator Panel with books, users, profiles and borrowed books
  • Page to login
  • Page to logout
  • Registration site
  • Paginations for books section
  • Profile accessibility for all new users
  • Custom views, functionalities and messages for logged in users
  • DB with a variety of books, imported from CSV file
  • Possibility for users to borrow books for a limited amount of time(i.e. 3 months)
  • Possibility for users to return books
  • Book counter for library and custom counter for users
